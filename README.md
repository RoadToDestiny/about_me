### Пример кода
```python
def calculate():
    return random.randint(1, 100)
    
@bot.message_handler(func=lambda message: message.text == 'Случайное число')
def handle_random_number(message):
    result = calculate()
    bot.send_message(message.chat.id, f'Ваше число: {result}')
```
